# FRAPPY
a framework for programming sample environment control nodes with a SECoP interface

soon you will find here a python framework allowing to create easily
a SECoP sample envionment control node.
This may be done either from scratch, including writing the drivers.
Or by enhancing existing software with a SECoP interface.

In the meantime you have a peek on what's coming here:
https://forge.frm2.tum.de/cgit/cgit.cgi/sine2020/secop/playground.git/tree/

Development is still going on, if you want to participate, try:
https://forge.frm2.tum.de/review/q/project:sine2020%252Fsecop%252Fplayground
(you'll need an account for accessing forge.frm2.tum.de, ask björn pedersen about it: mailto://bjoern.pedersen@frm2.tum.de)

current releases are available as .deb packages here:
https://forge.frm2.tum.de/repos/buster/

Or you just clone the git from here:
https://forge.frm2.tum.de/review/sine2020/secop/playground

cheers.

the frappy authors.
